📖 Cara Setup
🚀

Panduan Konfigurasi

Ikuti langkah berikut untuk menghubungkan ke GitHub

1
Buat repository baru di GitHub (boleh private atau public).
2
Buat file data.json di root repo dengan isi minimal: {"kategori":[],"artikel":[]}
3
Pergi ke Settings → Developer Settings → Personal Access Tokens → Tokens (classic), klik Generate new token.
4
Centang scope repo (full control of private repositories), lalu generate dan salin token.
5
Isi form di atas dengan username, nama repo, dan token kamu. Klik Simpan Konfigurasi, lalu Tes Koneksi.
